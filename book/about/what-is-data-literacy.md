# What is Data Literacy?

Data literacy defines how competent an individual is in collecting, analysing, interpreting, 
evaluating, storing, sharing, and communicating data, while bearing ethical, integrity, 
privacy, and security considerations. 

> A data literate individual should has the knoweldge and skills to collect, analyse, interpret, 
evaluate, store, share, and communicate data, while bearing ethical, integrity, 
privacy, and security considerations in mind.
